# 마음 연구소 — 프로젝트 소개

---

## 1. 과제 취지

현대인들은 심리적 어려움을 겪더라도 전문 상담에 대한 접근 장벽(비용, 시간, 사회적 낙인)으로 인해 도움을 받지 못하는 경우가 많다. **마음 연구소**는 임상심리학에서 검증된 심리 척도와 AI 대화 기술을 결합하여, 누구나 부담 없이 자신의 심리를 탐구하고 이해할 수 있는 환경을 제공하는 것을 목표로 기획되었다.

### 핵심 목표
- **자기 이해**: 성격, 애착 유형, 가치관, 스트레스 반응 패턴 등 다각적 심리 데이터를 통합하여 자기 인식 심화
- **접근성**: 전문 지식 없이도 사용 가능한 UI/UX와 쉬운 언어로 심리학의 대중화
- **지속성**: 감정 일지, AI 상담 기록 저장을 통해 시간에 따른 심리 변화 추적
- **안전성**: 진단이 아닌 심리 교육과 자기 탐구 도구로 명확히 제한

## 2. 사용 도구 및 기술

### 프레임워크
| 도구 | 버전 | 용도 |
|------|------|------|
| **Streamlit** | 1.45.0 | 웹 앱 UI 구축 (Python 기반 인터랙티브 웹) |
| **OpenAI API** | gpt-4o-mini | AI 심리 상담, 감정 분석, 성격 해석 생성 |
| **Plotly** | 6.0.1 | Big Five 레이더 차트, 감정 흐름 라인 차트, 태그 빈도 바 차트 |
| **pandas** | - | 데이터 처리 |
| **python-dotenv** | - | API 키 환경변수 관리 |

### 심리 척도 (검증된 임상 도구)
| 척도 | 원저자 | 측정 내용 |
|------|--------|-----------|
| **BFI-44** | John & Srivastava (1999) | 성격 5요인 (개방성·성실성·외향성·친화성·신경증) |
| **ECR-R** | Fraley et al. (2000) | 성인 애착 유형 (안정/불안/회피/두려움) |
| **Schwartz 가치 이론** | Schwartz (1992) | 핵심 가치관 10가지 |
| **Keirsey 기질 이론** | Keirsey (1978) | 기질 4유형 (SJ/SP/NF/NT) |
| **에니어그램** | Riso & Hudson | 성격 9유형과 핵심 동기 |

### 데이터 저장
- `mindlab_data.json` — 로컬 JSON 파일 (프로필, 검사 결과, 채팅 기록, 감정 일지 영구 저장)

### 개발 환경
- **언어**: Python 3.x
- **패키지 관리**: Conda (base 환경)
- **버전 관리**: Git / GitHub (`https://github.com/SarahCho0/MindLab`)

## 4. 앞으로의 개선 방향

### 단기 (1~2개월)

#### 1. 마음챙김 호흡 타이머
- 4-7-8 호흡법, 박스 호흡 등 시각적 애니메이션 가이드 추가
- Streamlit의 `st.empty()`와 JavaScript 인터페이스 활용
- 호흡 세션 완료 후 감정 전/후 비교 기록

#### 2. 인지 왜곡 체크리스트 (CBT)
- Beck의 인지 왜곡 10가지 기반 자동적 사고 기록지
- "상황 → 자동적 사고 → 감정 → 대안적 사고" 4단계 CBT 워크시트
- AI가 인지 왜곡 패턴을 분석하여 재구조화 제안

#### 3. 다중 사용자 지원
- 현재는 단일 `mindlab_data.json`에 한 명의 데이터만 저장
- 사용자별 프로필 분리 또는 클라우드 저장소(Firebase 등) 연동

### 중기 (3~6개월)

#### 4. 데이터 분석 대시보드
- 월별/주별 감정 흐름 비교
- 감정과 요일/시간대 상관 분석
- Big Five 점수 변화 추적 (재검사 기능 추가)

#### 5. 목표 설정 및 실천 트래킹
- 심리 성장 목표 설정 (예: "불안 조절 연습 21일")
- 매일 체크인 시스템과 AI 격려 메시지
- 목표 달성률 시각화

#### 6. 관계 패턴 분석
- 대인관계 스트레스 특화 상담 플로우
- 갈등 상황 롤플레이 연습 (AI 상대방 역할)
- 애착 유형 기반 소통 방식 코칭

### 장기 (6개월 이상)

#### 7. 임상 전문가 연계
- 심리 데이터 요약 리포트 PDF 생성 (전문 상담사에게 제공 가능)
- 전문 심리 상담 기관 정보 연동 (지역별 검색)

#### 8. 모바일 최적화 / PWA
- 모바일 친화적 레이아웃 전면 재설계
- Progressive Web App으로 홈 화면 설치 가능

#### 9. LLM 고도화
- Fine-tuning된 심리 상담 특화 모델 적용 (예: gpt-4o)
- 장기 대화 맥락 기억 시스템 (Vector DB 활용, LangChain 등)
- 감정 분석 NLP 모델 추가 (텍스트에서 감정 자동 태깅)

## 5. 윤리적 고려사항 및 한계

- 본 앱은 **심리 교육 및 자기 탐구 도구**이며, 임상 진단을 대체하지 않습니다.
- 정신건강 위기 상황에서는 전문가 도움을 안내합니다. (1393, 1577-0199)
- 수집된 데이터는 로컬에만 저장되며 외부 서버로 전송되지 않습니다.
- AI 응답은 참고용이며, 전문 임상의의 판단을 대체할 수 없습니다.
